# FWU CSIT Notes

A simple and lightweight website created to help **BSc CSIT students of Far-Western University** easily access important academic resources.

The website provides:
- **Notes Link**
- **Old question papers Link**
- **Syllabus Link**

All study materials are organized through a Google Drive folder for easy access.

---

## 🌐 Live Website:  
Access the website using the link below:  
https://fwu-csit-notes.vercel.app/

---

## 📌 Features
The website is kept as simple as possible to ensure easy access by all students.  
- 📚 **Notes Page**
  - Direct access to:
    - Lecture Notes
    - Old Question Papers
    - Syllabus
  - All Materials are stored in a Google Drive folder.

- ⚡ **Simple and Fast**
  - Built with only **HTML and CSS**
  - No frameworks or heavy dependencies
  - Lightweight and fast loading

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
